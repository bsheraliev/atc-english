# Лидерборд через Google Apps Script (как в «Ремонте»)

Бесплатно, без секретов в сайте: скрипт работает от вашего имени и пишет в вашу Google Таблицу.

1. Создайте новую **Google Таблицу** (sheets.new).
2. В ней: **Extensions → Apps Script**. Удалите шаблон, вставьте код из `leaderboard.gs`, сохраните.
3. **Deploy → New deployment** → шестерёнка → тип **Web app**:
   - *Execute as:* **Me**
   - *Who has access:* **Anyone**
   - **Deploy** → авторизуйте (это ваш аккаунт; разрешаете скрипту писать в вашу Таблицу).
4. Скопируйте **Web app URL** (вид `https://script.google.com/macros/s/…/exec`) и **пришлите мне** — он публичный, не секрет.
5. Я впишу его в `LEADERBOARD.scriptUrl` в `app.js` и опубликую. Кнопка «🏆 Лидерборд» начнёт показывать топ-30 и «Отправить мой результат».

**Заметки.** Имя берётся из Telegram (если открыто в боте) или «Гость». Хранится максимум XP на имя. Это казуальный лидерборд — без жёсткой защиты от накрутки. Если потом захотите менять код скрипта — после правок снова **Deploy → Manage deployments → Edit → New version**.
