# AvEng — режим «витрина»: как подключать игры

Сейчас в каждой группе (роли) **активны первые 3 плитки**, остальные помечены
бейджем **🔧 в разработке**: по нажатию игра не запускается, показывается тост
*«Раздел на стадии разработки — скоро будет доступен»* (рус/тадж/англ).

Активны по умолчанию (первые 3 в каждой роли):
- **Диспетчер / Пилот:** Радиоалфавит · Фразеология · Стандартные слова
- **Бортпроводник:** Радиоалфавит · Бортпроводник·сервис · Авиатермины
- **Сотрудник аэропорта:** Радиоалфавит · Аэропорт·пассажиры · Авиатермины

---

## Где ручки

Файл **`app.js`**, вверху, сразу под списком `MENU`:

```js
const WIP_AFTER = 3;     // сколько плиток активно в начале каждой группы
const LIVE_EXTRA = [];   // id игр (go), всегда активных независимо от позиции
```

## Как подключить позже

**1. Открыть конкретную игру (точечно, рекомендуется).**
Добавь её id (`go`) в `LIVE_EXTRA`. Пример — открыть «Read-back» и «Аварийные процедуры»:
```js
const LIVE_EXTRA = ["quiz-readback", "quiz-emergency"];
```
Игра станет активной во всех ролях, где она есть, даже если стоит ниже 3-й позиции.

**2. Открыть больше плиток сразу во всех группах.**
Подними число: `const WIP_AFTER = 5;` → активны первые 5 в каждой группе.

**3. Открыть всё меню целиком.**
```js
const WIP_AFTER = 99;
```

## После любой правки

1. В **`sw.js`** подними версию кэша: `const CACHE = "atc-eng-v2";` → `"atc-eng-v3";`
   (чтобы установленные PWA подхватили обновление).
2. Опубликуй: запусти **`Опубликовать.cmd`** (или `Автопубликация.cmd`).
   Сайт обновится за ~1 минуту: https://bsheraliev.github.io/atc-english/
3. На устройстве закрой и заново открой приложение (иногда дважды) — service
   worker отдаст новую версию.
