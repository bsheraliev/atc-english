# AvEng — статистика использования (GoatCounter)

Аналитика встроена, но **по умолчанию выключена**. Один раз подключаешь бесплатный
счётчик — и видишь визиты и поведение внутри игры в одном дашборде.

## Что считается
- **Визиты** — каждый запуск приложения (просмотр), уникальные пользователи, страны, источники переходов.
- **События внутри игры** (вкладка *Events* в дашборде):
  - `app-open` — запуск приложения;
  - `open/<id>` — какой модуль открыли (напр. `open/quiz-readback`, `open/elpet`);
  - `done/<категория>` — завершённое прохождение теста.

## Как включить (один раз, ~2 минуты)
1. Зайди на **https://www.goatcounter.com/** → *Sign up*. Бесплатно, без карты.
2. Придумай код сайта, например **`aveng`** — адрес дашборда станет `https://aveng.goatcounter.com`.
3. Открой **`app.js`**, найди строку:
   ```js
   const ANALYTICS = { goatcounter: "" };
   ```
   и впиши свой код:
   ```js
   const ANALYTICS = { goatcounter: "aveng" };
   ```
4. Подними версию кэша в **`sw.js`** (`atc-eng-v3` → `atc-eng-v4`) и опубликуй (`Опубликовать.cmd`).

## Где смотреть
Дашборд: `https://<твой-код>.goatcounter.com` (вход под твоим аккаунтом).
- *Pages* / общий график — визиты и уникальные.
- *Events* — действия внутри игры (что открывают, что проходят).
- Можно ставить период (день/неделя/месяц), смотреть страны и источники.

## Приватность
GoatCounter не использует cookies и не собирает персональные данные — баннер согласия не нужен.
Данные видны только тебе. Если код в `ANALYTICS` пустой — ничего не отправляется.
